БЕСПЛАТНЫЕ ВАРИАНТЫ ДЛЯ РАЗВЕРТЫВАНИЯ
=====================================

ВАРИАНТ 1: RAILWAY (РЕКОМЕНДУЕТСЯ - САМЫЙ ПРОСТОЙ)
---------------------------------------------------
✅ Бесплатно: $5 кредитов в месяц (достаточно для небольшого проекта)
✅ Автоматическое развертывание из GitHub
✅ Не нужно настраивать Docker вручную
✅ HTTPS из коробки

Как использовать:
1. Зарегистрируйтесь: https://railway.app
2. Подключите GitHub репозиторий
3. Railway автоматически определит проект и развернет
4. Добавьте переменные окружения в настройках проекта
5. Готово!


ВАРИАНТ 2: RENDER
-----------------
✅ Бесплатно: 750 часов в месяц
✅ Автоматическое развертывание
✅ HTTPS включен

Как использовать:
1. Зарегистрируйтесь: https://render.com
2. Создайте "New Web Service"
3. Подключите GitHub
4. Выберите ваш репозиторий
5. Настройте переменные окружения
6. Deploy!


ВАРИАНТ 3: FLY.IO
-----------------
✅ Бесплатно: 3 shared-cpu VMs
✅ Хорошая производительность
✅ Простое развертывание

Как использовать:
1. Зарегистрируйтесь: https://fly.io
2. Установите flyctl: curl -L https://fly.io/install.sh | sh
3. fly launch
4. Следуйте инструкциям


ВАРИАНТ 4: ORACLE CLOUD (БЕСПЛАТНЫЙ VPS)
-----------------------------------------
✅ Бесплатно навсегда: 2 VMs с 1GB RAM каждая
✅ Полный контроль
✅ Можно установить Docker

Как использовать:
1. Зарегистрируйтесь: https://www.oracle.com/cloud/free/
2. Создайте VM (Always Free tier)
3. Следуйте инструкциям из DEPLOY_STEPS.txt


ВАРИАНТ 5: GOOGLE CLOUD PLATFORM
---------------------------------
✅ Бесплатно: $300 кредитов на 90 дней
✅ После этого можно использовать Always Free tier
✅ Хорошая производительность

Как использовать:

ВАРИАНТ 6: AWS FREE TIER
-------------------------
✅ Бесплатно: 12 месяцев для новых пользователей
✅ EC2 t2.micro instance
✅ 750 часов в месяц

Как использовать:
1. Зарегистрируйтесь: https://aws.amazon.com/free

3. Следуйте инструкциям из DEPLOY_STEPS.txt


РЕКОМЕНДАЦИЯ
------------
Для начинающих: Railway или Render
- Не нужно настраивать сервер
- Автоматическое развертывание
- HTTPS из коробки
- Просто добавить переменные окружения

Для более продвинутых: Oracle Cloud
- Бесплатно навсегда
- Полный контроль
- Нужно настроить вручную (но есть инструкции)


ВАЖНО
-----
Все эти варианты требуют:
- GitHub репозиторий (или загрузить код вручную)
- Настроить переменные окружения (.env)
- Ваш Groq API ключ уже есть, так что это не проблема
